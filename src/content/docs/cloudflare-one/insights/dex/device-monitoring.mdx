In device monitoring, you will be able to review network and device performance data side by side with the information found
in [Fleet status](/cloudflare-one/insights/dex/fleet-status/).

Network and device performance data helps IT administrators to troubleshoot performance issues,
investigate network connectivity problems, and monitor device health.

## View a device's network performance

To view a device's network performance metrics:

1. Log in the [Cloudflare dashboard](https://dash.cloudflare.com/) and select your account.
2. Select **Zero Trust** to go to the Zero Trust dashboard.
3. Select **My team** > **Devices**.
4. Select a device > **View details** > select the **DEX** tab > scroll down to **Network performance**.
